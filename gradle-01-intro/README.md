## Hello gradle


Solo inicia una tarea con gradle desde el archivo 'build.gradle', de forma alternativa se agrega su alternativa en kotlin 'build.gradle.kts'

desde consola usar:

```
gradle hello
```

Resultado:
``` 
> Task :hello
    Hello gradle
```
